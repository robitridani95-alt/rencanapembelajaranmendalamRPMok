Konfigurasi Awal

Hubungkan aplikasi ke Google Apps Script backend

1
Deploy Code.gs sebagai Web App
  1. Buka Google Apps Script → paste Code.gs
  2. Klik Deploy → New Deployment
  3. Pilih tipe: Web App
  4. Execute as: Me | Access: Anyone
  5. Klik Deploy → copy URL-nya
2
Paste URL Web App di sini
💡 URL disimpan di browser (localStorage). Tidak perlu input ulang kecuali ganti perangkat.
